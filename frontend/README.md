# ScropIDS Frontend

Production-oriented SOC console frontend:

- React + Vite + TypeScript
- TailwindCSS + shadcn-style UI components
- React Router + Axios
- Recharts dashboards
- Dark SOC theme

## Local Dev

```bash
npm install
npm run dev
```

Environment:

- `VITE_API_BASE_URL` default is `/api/v1`
